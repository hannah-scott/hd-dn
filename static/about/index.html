HD-DN

HD-DN is my personal website, and my main online presence. I was inspired by seeing James Clear's book summaries and Hundred Rabbits' beautiful website.

This site is a combination of HTML and Go templates — I am generating things dynamically where I can. I don't use JS, and I don't use trackers. It is served by a Go server sat on a cloud machine: I assume my cloud service provider does some traffic logging.

The header font is Outward by Raoul Audouin. Distributed by velvetyne.fr.

The body font is Amaranth by Gesine Todt.

The monospace font is NotCourierSans, distributed by OSP.

I don't exert copyright on anything I create for this site. All materials — code, text, images — that I create for HD-DN are in the public domain.
