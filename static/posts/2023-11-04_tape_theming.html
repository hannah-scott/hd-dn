I love the idea of Tape. There's some issues I have with the workflow — like, I wish data was stored in plaintext1 rather than a SQLite DB — but conceptually and visually it ticks a lot of boxes for me.

You know what else I love? 100r's old SVG theming. Portable cross-app theming is a really neat idea — and showing themes as SVGs is clean as hell.

Luckily for me, Tape supports 100r theming!

Default theme colors

Based on reading the template in Aeriform's Github and messing around with SVG files manual testing, this seems to be how Tape uses 100r theming.

Groove is the Tape term for buttons that add things, i.e. the collapsible New Collection and Add buttons.

The only odd one out is tape_style, which is a style-setting variable with the following options:

  • '0' — status-specific background color
  • '1' — status-specific text color
  • '2' — status-specific text color with low saturation status-specific background color
  • '3' — status-specific text color, b_high-color border
  • '4' — status-specific text color and border
  • anything else — same as '0'
