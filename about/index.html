About me

I'm a big fan of making things. I'm a bigger fan of breaking things, forgetting to fix them and moving on to something else. My current areas of focus are trying to practice a more ethical style of interpersonal relationships, and testing out mindful approaches to personal tech.

HD-DN

HD-DN is my personal website, and my main online presence. I was inspired by seeing James Clear's book summaries and Hundred Rabbits' beautiful website.

This site is built with a static site generator I wrote in Python as a personal project, which I call ilo lipu. It's based off of cite, the original build tool for this site that I wrote in C.

I'm in the process of hand-rewriting the website, because I enjoy doing it and I want to learn more about good HTML practices.

Font

This site uses a self-hosted copy of Atkinson Hyperlegible.
